# Trabajo Práctico N°2 - Taller de Lenguajes (Python)
Este repositorio contiene la resolución de la segunda práctica de la materia.

## Estructura del Proyecto
El proyecto sigue una estructura modular:

- **`src/`**: Contiene los módulos `.py` con todas las funciones realizadas
- **`notebooks/`**: Contiene los archivos `.ipynb` donde se importan las funciones y se ejecutan los casos de prueba y ejercicios principales.
- **`README.md`**: Instrucciones de uso y descripción del proyecto.

## Instalación y Requisitos
Para ejecutar este proyecto, es necesario tener instalado **Python 3.x**. No se requieren librerías externas adicionales (se utilizaron módulos de la biblioteca estándar como `random`, `collections` y `math`).
